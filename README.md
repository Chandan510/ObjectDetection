# 🎯 Live Object Detection Using YOLO, Django, and React

A full-stack web application that uses **YOLO (You Only Look Once)** to detect objects **in real-time** from your device's camera.  
The backend (Django) runs the YOLO model and streams frames to the frontend (React), displaying the detected **object names** alongside the live video.

---

## 🚀 Features

- **Live camera streaming** from your browser
- **Real-time object detection** using YOLO
- **Displays detected object names** (no file uploads)
- **WebSocket/REST API integration** for smooth frame transfer
- **Cross-platform**: Works on Windows, macOS, and Linux
